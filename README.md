# 🚀 Idea Audit

Idea Audit is a simple tool to evaluate and validate ideas using structured scoring criteria.  
It helps founders, developers, and creators make better decisions before investing time and resources.

---

## 📌 Features

- 🧠 Structured idea evaluation
- 📊 Scoring system for decision making
- ⚙️ Customizable criteria
- 🔍 Quick and repeatable analysis
- 💡 Useful for startups, projects, and product ideas

---

## 🏗️ Use Cases

- Startup idea validation
- Product feature prioritization
- Hackathon project selection
- Business strategy evaluation

---

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/Ramasaikiran/Idea-audit.git
cd Idea-audit
````

Install dependencies:

```bash
pip install -r requirements.txt
```

---

## ▶️ Usage

Run the program:

```bash
python main.py
```

Example:

```text
Enter your idea: AI resume analyzer
Problem clarity (1-10): 8
Market demand (1-10): 7
Feasibility (1-10): 9
Competition (1-10): 6
Monetization (1-10): 7
```

Output:

```text
Final Score: 7.4 / 10
```

---

## 🧪 How It Works

1. Input your idea
2. Score different criteria
3. System calculates weighted result
4. Get a final evaluation score
